# app > dateTime

Path: metadata > app > dateTime.

Date-time definitions.

Example:

```json
{
    "dateFormatList": [
        "DD.MM.YYYY",
        "MM/DD/YYYY",
        "DD/MM/YYYY",
        "YYYY-MM-DD",
        "DD. MM. YYYY"
    ],
    "timeFormatList": [
        "HH:mm",
        "hh:mma",
        "hh:mmA",
        "hh:mm A",
        "hh:mm a"
    ]
}
```

## dateFormatList

*string[]*

A list of date formats available in the system.

## timeFormatList

*string[]*

A list of time formats available in the system.
